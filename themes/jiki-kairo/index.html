    🧲 磁気回路¶

    📍 学習マップ上の現在地

    前提 → 電磁力　→　[磁気回路]（現在地）　→　次 → インダクタンス

    重要度 頻出論点 バージョン
    S 磁束 / 磁気抵抗 / アンペアの法則 / ヒステリシス v0.7 ✅

    磁束が閉じた経路を流れる回路。電気回路と構造が1:1対応するため、「磁気のオームの法則」として解析できる。空隙（エアギャップ）の扱いが試験の核心。

    🧠 原理（なぜ起きるか）¶

    • 磁気回路は電気回路の完全なアナロジー。起磁力 \(F = NI\) が電圧源、磁束 \(\phi\) が電流、磁気抵抗 \(R_m\) が抵抗に対応する。
    • 「磁気のオームの法則」：\(\phi = F / R_m = NI \cdot \mu A / l\)。コイルの巻き数 \(N\) と電流 \(I\) の積が磁界を駆動する「圧力」になる。
    • 鉄心は透磁率が空気の数百〜数千倍。==空隙（エアギャップ）は細くても磁気抵抗が支配的==になる。

    5秒で思い出す

    電気回路の \(V = IR\) をそのまま磁気に置き換える。 \(NI = \phi \cdot R_m\)（起磁力 = 磁束 × 磁気抵抗）。エアギャップは「高抵抗の短い区間」。

    📐 公式（どう計算するか）¶

    レイヤーA：基本概念¶

    公式 意味（日本語） 使える条件 使えない条件
    \(F = NI\) [A] 起磁力。コイルの巻き数と電流の積 常に成立 —
    \(R_m = \dfrac{l}{\mu A}\) [H⁻¹] 磁気抵抗。長さ大・断面積小・透磁率低で増大 均一断面の直線磁路 漏れ磁束あり・テーパ形状
    \(\phi = \dfrac{F}{R_m} = \dfrac{NI \cdot \mu A}{l}\) 磁気のオームの法則。磁束は起磁力に比例し磁気抵抗に反比例 閉磁路・漏れ磁束無視 漏れ磁束が大きい場合

    レイヤーB：応用変換¶

    公式/手法 意味 使える条件 使えない条件
    \(B = \mu H = \mu_0 \mu_r H\) [T] 磁束密度 = 透磁率 × 磁界強度 均一・等方性材料 磁気飽和領域
    \(\phi = B \cdot A\) 磁束 = 磁束密度 × 断面積 均一磁束 不均一磁界
    キルヒホッフ起磁力則 \(\sum NI = \sum \phi R_m\) 閉じた磁路に沿った起磁力の和 = 磁気抵抗降下の和 閉磁路 —
    エアギャップの \(R_{m,gap} = \dfrac{l_g}{\mu_0 A}\) 空隙では \(\mu_r = 1\)（空気の透磁率は \(\mu_0\) のみ） ギャップ長 \(l_g\) が断面積に対し小さい場合 ギャップが大きく辺端効果あり

    📊 比較表¶

    1. 電気回路 vs 磁気回路（1:1対応）¶

    電気回路 磁気回路 単位
    起電力 \(V\) 起磁力 \(F = NI\) [V] vs [A]
    電流 \(I\) 磁束 \(\phi\) [A] vs [Wb]
    抵抗 \(R\) 磁気抵抗 \(R_m\) [Ω] vs [H⁻¹]
    電流密度 \(J\) 磁束密度 \(B\) [A/m²] vs [T]
    電界強度 \(E\) 磁界強度 \(H\) [V/m] vs [A/m]
    導電率 \(\sigma\) 透磁率 \(\mu\) [S/m] vs [H/m]
    オームの法則 \(V = IR\) 磁気オーム \(NI = \phi R_m\) —

    2. 鉄心 vs 空気の透磁率¶

    材料 透磁率 \(\mu_r\)（比透磁率） 磁気抵抗
    真空・空気 \(\mu_0 = 4\pi \times 10^{-7}\) H/m 1 基準（大きい）
    珪素鋼板 \(\mu_0 \times (1000\sim5000)\) \(1000\sim5000\) 空気の1/1000以下
    電磁軟鉄 \(\mu_0 \times (2000\sim8000)\) \(2000\sim8000\) さらに低い

    なぜエアギャップの磁気抵抗が鉄心より圧倒的に大きいのか（R05上問3・H26問3で頻出）

    磁気抵抗の式 \(R_m = \dfrac{l}{\mu A}\) で考えると：

    • 鉄心 \(l_i = 0.5\,\text{m}\)、\(\mu_r = 2000\)：\(R_{m,\text{iron}} = \dfrac{0.5}{\mu_0 \times 2000 \times A}\)
    • 空隙 \(l_g = 1\,\text{mm} = 0.001\,\text{m}\)、\(\mu_r = 1\)：\(R_{m,\text{gap}} = \dfrac{0.001}{\mu_0 \times 1 \times A}\)

    比率を計算すると：

    \[\frac{R_{m,\text{gap}}}{R_{m,\text{iron}}} = \frac{0.001}{0.5} \times \frac{2000}{1} = 0.002 \times 2000 = 4\]

    1 mmの空隙でも、0.5 mの鉄心よりも4倍も磁気抵抗が大きい。 試験で「起磁力のほとんどが空隙で消費される」という論説文が正解になる理由はここにある。

    3. 直列磁気回路 vs 並列磁気回路¶

    項目 直列磁気回路 並列磁気回路
    磁束 \(\phi\) 各区間で等しい（電気の直列電流と同様） 分岐する（起磁力が等しい）
    合成磁気抵抗 \(R_m = R_{m1} + R_{m2} + \cdots\)（和） \(1/R_m = 1/R_{m1} + 1/R_{m2} + \cdots\)（逆数の和）
    起磁力の配分 \(NI = \phi R_{m1} + \phi R_{m2}\)（各区間に分配） 全区間に同じ \(NI\) がかかる

    環状ソレノイドの計算手順（H20問3の典型パターン）

    環状鉄心にコイルを巻いた構造（環状ソレノイド）は直列磁気回路の代表例。計算の流れは以下の3ステップ：

    1. 起磁力を求める：\(F = NI\)
    2. 各区間の磁気抵抗を求める
      • 鉄心区間：\(R_{m,\text{iron}} = \dfrac{l_i}{\mu_0 \mu_r A}\)
      • 空隙区間：\(R_{m,\text{gap}} = \dfrac{l_g}{\mu_0 A}\)（\(\mu_r = 1\)）
    3. 磁束を求める：\(\phi = \dfrac{NI}{R_{m,\text{iron}} + R_{m,\text{gap}}}\)

    磁束密度が必要なら \(B = \phi / A\) で変換する。

    計算での注意点

    単位を統一すること。\(l_g\) が mm 単位で与えられても \(R_m\) の式では m に換算して代入する。 断面積 \(A\) は cm² ではなく m² に換算（\(1\,\text{cm}^2 = 10^{-4}\,\text{m}^2\)）。

    🕳️ よくある勘違いTOP3¶

    ❌ 1：磁気抵抗の計算で \(\mu_r\) を忘れて \(\mu_0\) だけで計算しがち

    ✅ 鉄心の透磁率は \(\mu = \mu_0 \mu_r\)。\(\mu_r\) は数千になることもある。鉄心区間では \(\mu_r\) を掛けることで磁気抵抗が大幅に下がる。 空隙のみ \(\mu_r = 1\)（空気）。区間ごとに材料を確認すること。

    ❌ 2：磁束 \(\phi\) と磁束密度 \(B\) を混同しがち

    ✅ \(B\) は単位面積あたりの磁束（密度）、\(\phi\) は断面全体を貫く磁束の総量。\(\phi = B \cdot A\) の関係。 断面積 \(A\) が変わると同じ \(\phi\) でも \(B\) が変化する。設問でどちらを求めているか確認。

    ❌ 3：エアギャップの磁気抵抗が小さいと思いがち

    ✅ ギャップは長さが数mm以下でも \(\mu_r = 1\) のため磁気抵抗が非常に大きい。==鉄心（\(\mu_r \sim 2000\)）と比べると同じ長さで2000倍の磁気抵抗==になる。 回路全体の磁気抵抗をギャップが支配する場合も多い。起磁力の大半がギャップで消費される。

    磁力線の特徴（R02問4・H25問3 論説問題の頻出論点）

    磁力線の性質は論説問題で繰り返し問われる。以下を区別して覚える。

    性質 内容
    閉じた曲線 磁力線はN極から出てS極に入り、内部でもつながる（途中で切れない・終わらない）
    交差しない 磁力線同士は互いに交わらない（交差点では方向が一意に定まらなくなるため）
    密度 = 強さ 磁力線の本数（密度）が高いほど磁界が強い
    電気力線との違い 電気力線は正電荷で発生して負電荷で終わる（閉じていない）。磁力線は必ず閉じる

    引っかかりやすいポイント：磁力線は「N極を出て空間を通りS極に入る」ように見えるが、磁石内部でもS→N方向につながっており全体として閉じている。

    ヒステリシスループの読み方（R01問3・H29問4・H27問3・H18問3）

    ヒステリシスループは「BH曲線を往復させたときに描かれる閉じたループ」で、穴埋め・論説問題で頻繁に出題される。

    B（磁束密度）
↑    ___
|   /   \   ← 磁界Hを増加させるとBが増加（上の経路）
|  /     \
--+-------+-- → H（磁界の強さ）
|  \     /
|   \___/   ← 磁界Hを減少させるとBが減少（下の経路、経路が異なる）

    押さえるべき用語と意味：

    用語 定義 試験での問われ方
    残留磁気 \(B_r\) \(H = 0\) にしてもBが残る値（縦軸との交点） 「磁界をゼロにしたときの磁束密度」
    保磁力 \(H_c\) Bをゼロに戻すのに必要な逆向きのHの大きさ（横軸との交点） 「磁束密度をゼロにするために必要な磁界の強さ」
    ヒステリシス損 ループの面積に比例したエネルギー損失（1サイクルごとに熱として失われる） 「ループ面積が大きいほど損失が大きい」

    軟磁性体 vs 硬磁性体の違い（穴埋め頻出）

    • 軟磁性体（珪素鋼板など）：ループが細い → 保磁力小・残留磁気小 → ヒステリシス損が小さい → 変圧器・電動機の鉄心に使う
    • 硬磁性体（永久磁石用合金など）：ループが太い → 保磁力大・残留磁気大 → 一度磁化すると磁気を保持しやすい → 永久磁石に使う

    🧩 正答者 vs 誤答者¶

    観点 ❌ 誤答者 ✅ 正答者
    磁気のオームの法則 電気回路のV=IRを機械的に当てはめてミスする F = φ × Rm（起磁力=磁束×磁気抵抗）を意識して適用
    透磁率の扱い 空気ギャップでもμrを使う 空気ギャップではμ₀のみ（μr=1）を使う
    φとBの区別 φとBを混同して面積Aを忘れる φ = B × A（磁束密度×断面積が磁束）を常に意識
    磁気抵抗の直並列 電気抵抗と同じ向きで合成する 電気抵抗と同じ計算方法（直列は和、並列は逆数の和）で合成
    起磁力の単位 「V（ボルト）」と書いてしまう 起磁力の単位は [A]（アンペア）または [A回]

    確認問題

    問題: 磁路長l=0.5m、断面積A=10cm²、透磁率μ=1000μ₀の鉄心コアの磁気抵抗Rmを求めよ。（μ₀=4π×10⁻⁷）

    解答

    答え: Rm = l/(μA) = 0.5/(1000×4π×10⁻⁷×10×10⁻⁴) ≈ 3.98×10⁵ [H⁻¹] ポイント: 磁気抵抗 = 磁路長 ÷（透磁率×断面積）。電気抵抗 R=l/(σA)と対応

    Level 2: 数学的背景 🔬

    BH曲線と磁気飽和

    実際の鉄心は線形ではなく、BH曲線（磁化曲線）に従う非線形特性を持つ。 - 初磁化曲線：未磁化状態からHを増やすとBが増加 - 磁気飽和：Hを増やしてもBがほとんど増えない領域 - ヒステリシス：HとBの関係が往復で異なる（ヒステリシス損の原因）

    電験3種の磁気回路問題は線形（μ = 一定）を仮定しているが、実際の変圧器・電動機設計では飽和領域を考慮したBH曲線の活用が必須。

    Level 3: 実務との接点 🏭

    変圧器の鉄心設計は磁気回路の直接応用。空隙（エアギャップ）の大きさで変圧器の磁気特性が変わる。現場では「変圧器の励磁電流が大きい」=「磁気回路のインピーダンスが高い」として理解する。

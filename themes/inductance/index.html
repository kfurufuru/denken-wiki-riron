    🌀 インダクタンス¶

    📍 学習マップ上の現在地

    前提 → 磁気回路　→　[インダクタンス]（現在地）　→　次 → 三相交流

    重要度 頻出論点 バージョン
    S 自己・相互インダクタンス / 結合係数 / エネルギー v0.7 ✅

    コイルは「電流の変化を嫌がる素子」。磁界にエネルギーを蓄え、電流変化に逆らう起電力を発生させる。

    🧠 原理（なぜ起きるか）¶

    • フライホイール（はずみ車）は回転の変化に抵抗する。インダクタも同じで、電流が急に増えようとすると逆起電力で抵抗し、電流が急に減ろうとすると維持しようとする。「電流の慣性」がインダクタンスの本質。
    • エネルギーの蓄え方が違う：コンデンサは==電界に蓄える==（電荷 → 電圧）、インダクタは==磁界に蓄える==（電流 → 磁束）。対になる概念として一緒に整理する。

    5秒で思い出す

    インダクタ ＝ フライホイール。「急に変えようとするな」と抵抗する。エネルギーは磁界の中に。

    📐 公式（どう計算するか）¶

    レイヤーA：基本概念¶

    公式 意味（日本語） 使える条件 使えない条件
    $\(e = -L\dfrac{dI}{dt}\)$ 電流が変化するとコイルは逆起電力を生じる 線形コイル、均一磁界 鉄心が磁気飽和している場合
    $\(L = \dfrac{N\phi}{I} = \dfrac{\mu N^2 A}{l} \text{ [H]}\)$ コイルの構造（巻数・断面積・長さ・透磁率）でLが決まる 均一磁界の場合 漏れ磁束が大きい場合は近似
    $\(W = \dfrac{1}{2}LI^2 \text{ [J]}\)$ コイルに蓄えられるエネルギーは電流の2乗に比例 電流が定常状態 過渡状態では瞬時値が変化

    環状鉄心コイルのL計算手順（R07上問3・H20問4・H18問4 出題）

    環状鉄心コイルの自己インダクタンス計算は出題頻度が高い。公式の各記号の意味を理解して代入できるようにしておく。

    \[L = \frac{\mu_0 \mu_r N^2 A}{l} \text{ [H]}\]
    記号 意味 単位 注意点
    \(\mu_0\) 真空の透磁率 \(4\pi \times 10^{-7}\) H/m 定数として与えられることが多い
    \(\mu_r\) 比透磁率（鉄心の材料で決まる） 無次元 問題文で「比透磁率＝○○」として与えられる
    \(N\) 巻数 回（無次元） Lは\(N^2\)に比例。巻数を2倍にするとLは4倍
    \(A\) 鉄心の断面積 m² cm²で与えられたら \(\times 10^{-4}\) に変換
    \(l\) 磁路の長さ（環状鉄心の中心線の周長） m cm で与えられたら \(\times 10^{-2}\) に変換

    「巻数とLの関係」が問われる典型問題

    巻数を \(N_1\) から \(N_2\) に変えたときのインダクタンス比：

    \[\frac{L_2}{L_1} = \left(\frac{N_2}{N_1}\right)^2\]

    例：巻数を2倍にした → \(L\) は4倍になる。巻数を半分にした → \(L\) は \(\frac{1}{4}\) になる。

    レイヤーB：応用変換¶

    公式/手法 意味 使える条件 使えない条件
    $\(e_2 = -M\dfrac{dI_1}{dt}\)$ コイル1の電流変化がコイル2に起電力を誘導（相互誘導） 2コイルが磁気的に結合している場合 磁気遮蔽されている場合
    $\(k = \dfrac{M}{\sqrt{L_1 L_2}} \quad (0 \leq k \leq 1)\)$ 結合係数：どれだけ磁束が共有されているか 常に定義可能 k=1は理想変圧器（漏れ磁束ゼロ）の理想値
    $\(L = L_1 + L_2 + 2M \quad \text{（和動）}\)$ 磁束が同方向に強め合う直列接続 直列かつ磁束が同方向 磁束が逆方向（差動）では符号反転
    $\(L = L_1 + L_2 - 2M \quad \text{（差動）}\)$ 磁束が逆方向に弱め合う直列接続 直列かつ磁束が逆方向 和動接続では符号反転

    和動・差動の見分け方と計算手順（H29問3・R04上問3 出題）

    合成インダクタンスの計算は「和動か差動か」の判別が最大の落とし穴。

    回路図から判別する手順

    1. 各コイルの同名端子（•印または ◇ 印）を確認する
    2. 2つのコイルを直列に接続したとき、電流の流れる向きと同名端子の位置を見る
    3. 「電流が同名端子側から入る・入る」または「同名端子側から出る・出る」→ 和動（＋2M）
    4. 「一方は同名端子から入り、他方は同名端子から出る」→ 差動（−2M）

    Mの値から最大合成インダクタンスを逆算する

    和動と差動のL測定値から M を求めることができる：

    \[M = \frac{L_{和動} - L_{差動}}{4}\]

    これを利用して「和動接続と差動接続のL測定値が与えられ、M または \(k\) を求めよ」という問題が出る。

    Mが求まったら k を計算：

    \[k = \frac{M}{\sqrt{L_1 L_2}}\]

    引っかかりやすいポイント：「直列接続だから単純に \(L_1 + L_2\)」と計算すると誤り。相互結合がある場合は必ず \(\pm 2M\) を加減する。

    📊 比較表¶

    コンデンサ vs インダクタ（双対関係）¶

    項目 コンデンサ（C） インダクタ（L）
    蓄えるエネルギー 電界 磁界
    エネルギー式 $\(W = \dfrac{1}{2}CV^2\)$ $\(W = \dfrac{1}{2}LI^2\)$
    急変を嫌うもの 電圧（V） 電流（I）
    直流定常状態 開放（電流ゼロ） 短絡（電圧ゼロ）
    インピーダンス $\(Z = \dfrac{1}{j\omega C}\)$ $\(Z = j\omega L\)$

    自己インダクタンス vs 相互インダクタンス¶

    項目 自己インダクタンス（L） 相互インダクタンス（M）
    対象 1つのコイル 2つのコイルの組み合わせ
    物理的意味 自分自身の電流変化に対する逆起電力 相手コイルの電流変化に対する誘導起電力
    単位 H（ヘンリー） H（ヘンリー）
    符号 常に正 接続方向で正負が変わる

    和動接続 vs 差動接続¶

    項目 和動接続（＋2M） 差動接続（－2M）
    磁束の向き 同方向（強め合う） 逆方向（弱め合う）
    合成インダクタンス $\(L_1 + L_2 + 2M\)$（大きくなる） $\(L_1 + L_2 - 2M\)$（小さくなる）
    巻き方の判断 同名端子を直列 異名端子を直列

    コンデンサとインダクタの直列・並列公式の違い（双対関係）

    コンデンサとインダクタは「双対」の関係にある。公式の形が入れ替わるので混同に注意。

    接続 コンデンサ C インダクタ L（相互結合なし）
    直列 \(\dfrac{1}{C} = \dfrac{1}{C_1} + \dfrac{1}{C_2}\)（合成容量が小さくなる） \(L = L_1 + L_2\)（合成Lが大きくなる）
    並列 \(C = C_1 + C_2\)（合成容量が大きくなる） \(\dfrac{1}{L} = \dfrac{1}{L_1} + \dfrac{1}{L_2}\)（合成Lが小さくなる）

    覚え方：コンデンサの直列とインダクタの並列が「同じ形の式」、コンデンサの並列とインダクタの直列が「同じ形の式」になる（完全に入れ替わっている）。

    🕳️ よくある勘違いTOP3¶

    ❌ こう思いがち①: 自己誘導起電力のマイナス符号は計算上の話で、物理的意味はない ✅ 実際は: マイナスはレンツの法則そのもの。「変化を妨げる向き」に起電力が生じることを示す。符号を落とすと向きが逆になり、エネルギーの授受が逆転する。

    ❌ こう思いがち②: 相互インダクタンスMは常に正の値 ✅ 実際は: Mの絶対値は常に正だが、合成インダクタンスを求める際は接続方向（和動か差動か）によって +2M か −2M になる。巻き方（同名端子の位置）で決まる。

    ❌ こう思いがち③: 結合係数k=1のコイルは現実に存在する ✅ 実際は: k=1は「漏れ磁束がゼロ」の理想変圧器の条件。現実のコイルは必ず漏れ磁束があるので k<1。k≈1 に近いのは鉄心トランスで、空芯コイルはkがかなり小さい。

    ❌ こう思いがち④: 相互結合コイル回路の電圧計算では、自己インダクタンス分の電圧だけ考えればよい ✅ 実際は: 相互結合がある場合、コイルの端子電圧は自己誘導（\(L \frac{dI}{dt}\)）と相互誘導（\(M \frac{dI'}{dt}\)）の両方を加減する必要がある。和動か差動かで符号が変わるため、向きの確認が必須。

    🧩 正答者 vs 誤答者¶

    観点 ❌ 誤答者 ✅ 正答者
    自己誘導の符号 e = L(dI/dt) とマイナスを忘れる e = -L(dI/dt)（逆起電力はレンツの法則でマイナス）
    和動・差動の判別 接続図を見ずに「直列なら和」と思い込む 磁束が強め合う向き（和動）か打ち消し合う向き（差動）かで符号が変わる
    結合係数kの範囲 k > 1 になることがあると思い込む 0 ≤ k ≤ 1（k=1は理想的な完全結合、現実はk<1）
    エネルギーの公式 W = LI（Iの2乗を忘れる） W = LI²/2（運動エネルギーのmv²/2と対比して覚える）
    インダクタの電流 スイッチ切替時に電流が急変すると考える インダクタの電流は急変できない（過渡現象の初期条件）

    相互結合コイル回路の電圧最大値の求め方（R05上問10 出題）

    「コイルを接続した回路に現れる電圧の最大値を求めよ」という計算問題の解法。

    問題パターン: 2つのコイル \(L_1\)、\(L_2\)（相互インダクタンス \(M\)）が和動または差動で直列接続された回路に交流電流 \(i = I_m \sin\omega t\) が流れるとき、回路の端子電圧の最大値を求める。

    解法手順

    1. 合成インダクタンスを計算する
    2. 和動：\(L = L_1 + L_2 + 2M\)
    3. 差動：\(L = L_1 + L_2 - 2M\)
    4. 誘導リアクタンスを計算する：\(X_L = \omega L\)
    5. 電流最大値 \(I_m\) と \(X_L\) を掛けて電圧最大値を求める：\(V_m = X_L \cdot I_m = \omega L \cdot I_m\)

    なぜ \(V_m = \omega L \cdot I_m\) になるか

    交流電流 \(i = I_m \sin\omega t\) をコイルに流すと、電圧は：

    \[v = L \frac{di}{dt} = L \cdot I_m \omega \cos\omega t\]

    最大値は \(\cos\omega t = 1\) のとき → \(V_m = \omega L I_m\)

    引っかかりやすいポイント：問題に \(M\) が与えられると「どう使うのか？」で迷いがちだが、まず合成インダクタンスを作って、あとは単一コイルと同じ手順で計算する。

    確認問題

    問題: L=50mHのコイルに5Aの電流が流れているとき、蓄えられているエネルギーを求めよ。

    解答

    答え: W = LI²/2 = 0.05×25/2 = 0.625J ポイント: W = LI²/2（運動エネルギー mv²/2 との対応で覚える）

    Level 2: 数学的背景 🔬

    結合インダクタの等価回路

    相互インダクタンスMを持つ2コイルの等価回路： - T型等価回路（Y等価）：L1-M、L2-M、Mの3インダクタ - 理想変圧器の条件：k=1（完全結合）のとき

    \[ M = k\sqrt{L_1 L_2} \quad (0 \leq k \leq 1) \]

    変圧器の動作原理：1次コイルの磁束変化が2次コイルに相互誘導起電力を発生させる。巻数比 \( N_1/N_2 \) が電圧変換比、\( N_2/N_1 \) が電流変換比に対応。

    Level 3: 実務との接点 🏭

    変電所・工場の変圧器はインダクタンスの相互結合で電圧変換する。リアクトル（インダクタ）は系統の短絡電流制限や高調波フィルタに使う。現場では「インピーダンス %（%Z）」という形でインダクタンスを管理する。

# バージョニング基準

このWikiの各ページには、コンテンツの成熟度を示すバッジが付いています。
バッジを見ることで「どの程度信頼できるか」「どの程度完成しているか」が一目でわかります。

---

## バッジ定義

### <span class="version-badge v05">v0.5</span> AI生成+構造化済み

- **状態**: AIが生成したコンテンツを構造化・整形した段階
- **信頼性**: 教科書・公式資料での検証は未実施
- **マーカー**: ページ内に `⚠️ AI生成・未検証` の注意書きが付く
- **使い方**: 「まず全体像をつかむ」目的なら使える。計算に使う公式は要確認

### <span class="version-badge v07">v0.7</span> 教科書・Web検証済み

- **状態**: 教科書または公式Webサイトでの内容確認が完了した段階
- **信頼性**: ⚠️が✅に変わる。公式・手順の正確性を保証
- **マーカー**: ページ内の未検証部分は解消済み
- **使い方**: 試験勉強に使える。過去問との対応付けを開始

### <span class="version-badge v10">v1.0</span> 完成版

- **状態**: 以下の条件をすべて満たした完成版
  - [ ] 頻出公式一覧へのリンク完備
  - [ ] 過去問全問に対応するページ内リンク（Level 2/3）
  - [ ] 学習ロードマップへの組み込み完了
  - [ ] フェインマンセッション実施済み（理解度スコア4以上）
- **信頼性**: 最高。試験直前の確認にも使える
- **使い方**: このページで学んだ内容を「他人に教えられる」レベルを目指す

---

## バージョンアップの手順

```
placeholder → v0.5: AIドラフト生成 + 構造整形
v0.5 → v0.7: 教科書照合 + ⚠️解消 + 過去問リンク追加開始
v0.7 → v1.0: 全リンク完備 + フェインマン検証 + ロードマップ統合
```

---

## placeholder ステータス

`version: placeholder` は、ページは存在するがコンテンツが未作成の状態です。
制作予定はあるが優先度が低いか、別エージェントが担当予定のテーマに付与されます。

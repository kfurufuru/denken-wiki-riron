## 1. 三角関数の基本

### 基本値テーブル

| 角度 | \(\sin\theta\) | \(\cos\theta\) | \(\tan\theta\) |
|------|---------------|---------------|---------------|
| 0° | 0 | 1 | 0 |
| 30° | \(1/2\) | \(\sqrt{3}/2\) | \(1/\sqrt{3}\) |
| 45° | \(\sqrt{2}/2\) | \(\sqrt{2}/2\) | 1 |
| 60° | \(\sqrt{3}/2\) | \(1/2\) | \(\sqrt{3}\) |
| 90° | 1 | 0 | 未定義 |

### よく使う公式

\[ \sin^2\theta + \cos^2\theta = 1 \]

\[ \tan\theta = \frac{\sin\theta}{\cos\theta} \]

\[ \sin(90° - \theta) = \cos\theta, \quad \cos(90° - \theta) = \sin\theta \]

### cosを基準とした場合のsinとの関係

- \(\sin\theta = \cos(90° - \theta)\): sinはcosより**90°遅れ**
- 交流回路では「電圧を基準（0°）」に取り、電流の位相差を\(\varphi\)で表す

!!! example "電験での使いどころ"
    力率角 \(\varphi\) の計算: \(\cos\varphi = R/Z\)。インピーダンス三角形で右図の斜辺=Z、底辺=R、高さ=\(X_L - X_C\)。

## 2. ベクトルと位相（フェーザー）

### フェーザー表現

交流電圧・電流は**大きさと位相角**で表す:

\[ \dot{V} = V\angle\theta \quad \text{（大きさ } V \text{、位相角 } \theta \text{）} \]

### 極形式 → 直交形式（成分分解）

\[ A\angle\theta \rightarrow (A\cos\theta,\ A\sin\theta) \]

実部: \( x = A\cos\theta \)、虚部: \( y = A\sin\theta \)

### 直交形式 → 大きさ（合成）

\[ |A| = \sqrt{x^2 + y^2} \]

### フェーザー図の描き方

```
1. 基準ベクトル（電圧または電流）を右向き水平に描く
2. 進み要素（容量性）は上向き（+j方向）
3. 遅れ要素（誘導性）は下向き（-j方向）
4. 合成は平行四辺形の対角線
```

!!! example "電験での使いどころ"
    RLC直列回路で各素子の電圧降下をベクトル合成し、電源電圧と電流の位相差（力率角）を求める問題で使う。

## 3. 複素数（交流回路の計算に必須）

### 複素数の形式

\[ z = a + jb \quad (j = \sqrt{-1}) \]

- \(a\): 実部（抵抗成分に対応）
- \(b\): 虚部（リアクタンス成分に対応）

### 絶対値と偏角

\[ |z| = \sqrt{a^2 + b^2}, \quad \arg(z) = \arctan\!\left(\frac{b}{a}\right) \]

### 極形式との変換

\[ z = a + jb = |z|\angle\theta = |z|(\cos\theta + j\sin\theta) \]

### 複素数の積・商

\[
z_1 \cdot z_2 = |z_1||z_2|\angle(\theta_1 + \theta_2) \quad \text{（大きさは積、角度は和）}
\]

\[
\frac{z_1}{z_2} = \frac{|z_1|}{|z_2|}\angle(\theta_1 - \theta_2) \quad \text{（大きさは商、角度は差）}
\]

### よく使う公式（暗記必須）

| 式 | 結果 |
|----|------|
| \( j^2 \) | \(-1\) |
| \( j^3 \) | \(-j\) |
| \( j^4 \) | \(1\) |
| \( 1/j \) | \(-j\)（= \(-j/j^2 = -j/(-1) = -j\)） |
| \( j \times (-j) \) | \(1\) |

!!! example "電験での使いどころ"
    インピーダンス \( Z = R + jX_L \) の形で表し、\(|Z|\) を計算して電流を求める。複素数の商（電圧÷インピーダンス）で電流の大きさと位相を一度に求められる。

## 4. 対数（dB計算に使う）

### log10の基本値

| 式 | 値 |
|----|----|
| \(\log_{10} 1\) | 0 |
| \(\log_{10} 10\) | 1 |
| \(\log_{10} 100\) | 2 |
| \(\log_{10} 1000\) | 3 |
| \(\log_{10} 2\) | 0.301（近似） |
| \(\log_{10} 3\) | 0.477（近似） |

### dBの計算

\[
\text{電圧比 [dB]} = 20\log_{10}\!\left(\frac{V_2}{V_1}\right)
\]

\[
\text{電力比 [dB]} = 10\log_{10}\!\left(\frac{P_2}{P_1}\right)
\]

!!! example "電験での使いどころ"
    電子回路の増幅器の利得（ゲイン）を dB で表す問題。「電圧が10倍になると +20 dB」「電力が10倍になると +10 dB」を暗記しておくと素早く解ける。

## 5. 連立方程式（キルヒホッフ法で必要）

### 2元連立方程式（クラメールの公式）

\[
\begin{cases} a_1 x + b_1 y = c_1 \\ a_2 x + b_2 y = c_2 \end{cases}
\]

\[
x = \frac{c_1 b_2 - c_2 b_1}{a_1 b_2 - a_2 b_1}, \quad
y = \frac{a_1 c_2 - a_2 c_1}{a_1 b_2 - a_2 b_1}
\]

### 計算ミスを防ぐチェックポイント

```
1. 文字を立てる前に「何を求めるか」を明確にする
2. KVLは「起電力の和 = 電圧降下の和」の形に統一する
3. 連立方程式を立てたら、次元（単位）が揃っているか確認する
4. 解いた値を元の式に代入して検証する（答えの逆算）
5. 符号（+/-）のミスが最多。電流の向きを図に書き込んでから立式する
```

!!! example "電験での使いどころ"
    網目電流法や節点電位法でKVL/KCLを立てると2〜3元の連立方程式になる。3元以上は行列式より「代入消去法」が速い場合が多い。

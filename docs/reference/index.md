# リファレンス

テーマ記事（各単元の解説ページ）と並行して使う参照ページ集。
単元を読みながら「公式の確認」「用語の意味確認」「典型問題のパターン確認」が必要になったときにすぐ開く。

## 試験直前の使い方

試験が近づいたら、以下の順でリファレンスを活用する。

### 2〜4 週間前: 定義の精度を上げる
- **用語集** を頭から通読し、「知っているつもり」の用語を洗い出す
- 特に「アドミタンス vs インピーダンス」「有効・無効・皮相電力の違い」など混同しやすいペアを確認する

### 1〜2 週間前: パターンを叩き込む
- **回路パターン集** の 10 パターンをそれぞれ1回手を動かして解く
- 「この回路形を見たらどのパターンか」の判断を反射的にできるよう練習する
- 過去問と照合して「どのパターンに当てはまるか」を確認する

### 3〜7 日前: 公式と単位を最終確認
- **頻出公式一覧** をざっと見直し、暗記が不確かな公式に付箋をつける
- **単位・次元チェック表** で単位の取り違えがないかを確認する

### 試験当日（直前）
- **回路パターン集** の「よくある計算ミス」の欄を読む（5 分でできる）
- **用語集** で苦手用語を 3〜5 語だけ最終確認する

---

> **使い方のコツ**: リファレンスページは「通読するページ」ではなく「辞書として引くページ」。問題を解いていて詰まったとき・わからない用語に出会ったときに開く。テーマ記事の方で概念を理解し、こちらでは定義・手順・パターンを確認するという使い分けが効果的。

# 分野別 過去問一覧

分野ごとに頻出問題タイプをまとめています。
詰まった問題の分野を確認して、対応テーマ記事で復習してください。

## 電気回路（直流）

**関連テーマ**: [直流回路](../themes/chokuryu-kairo.md)

頻出問題タイプ:

- キルヒホッフの法則（KVL/KCL連立方程式）
- テブナン等価回路による端子電圧・電流計算
- 直並列合成抵抗と分圧・分流
- 電力・電力量・効率の計算
- 抵抗温度係数による抵抗変化

→ [直流回路テーマ記事へ](../themes/chokuryu-kairo.md)

## 電磁気（毎年4〜5問・最重要）

**関連テーマ**: [静電気](../themes/seidenki.md), [コンデンサ](../themes/condenser.md), [電磁力](../themes/denjiryoku.md), [磁気回路](../themes/jiki-kairo.md), [インダクタンス](../themes/inductance.md)

| テーマ | 出題頻度 | 形式傾向 |
|---|---|---|
| 平行平板コンデンサ（電界・電束密度・誘電体挿入・静電エネルギー） | ほぼ毎年 | 計算・論説両方 |
| 電気力線の特徴 | 2〜3年に1回 | 論説・穴埋 |
| クーロンの法則・点電荷に加わる力 | 2〜3年に1回 | 計算 |
| 磁気回路・磁気抵抗・鉄心 | 2〜3年に1回 | 計算 |
| 磁界の強さ・アンペアの法則・ビオサバール | 2〜3年に1回 | 計算 |
| 電磁誘導・誘導起電力・フレミング | 2〜3年に1回 | 計算 |
| 帯電した導体球・電位・電荷 | 2〜3年に1回 | 計算 |
| ヒステリシスループ・磁化曲線 | 2〜3年に1回 | 穴埋 |
| 強磁性体の特性 | 2〜3年に1回 | 論説・穴埋 |
| 合成インダクタンス・相互インダクタンス | 不定期 | 計算 |

**学習の急所**: 平行平板コンデンサは形を変えてほぼ毎年出る。「電界E=V/d」「電束密度D=εE」「静電エネルギーW=QV/2=CV²/2」の3式を完全習得すること。

## 交流回路（毎年3〜4問・重要）

**関連テーマ**: [交流回路基礎](../themes/kouryu-kiso.md), [RLC回路](../themes/rlc-kairo.md), [交流電力](../themes/kouryu-denryoku.md)

| テーマ | 出題頻度 | 形式傾向 |
|---|---|---|
| 直列・並列共振回路（RLC） | ほぼ毎年 | 計算 |
| 交流電力・力率（有効・皮相・無効） | ほぼ毎年 | 計算 |
| ひずみ波交流（実効値・消費電力） | 2〜3年に1回 | 計算 |
| 波形率・波高率 | 不定期 | 穴埋 |
| インピーダンス計算・位相差 | 2〜3年に1回 | 計算 |
| LC回路の共振周波数大小比較 | 不定期 | 計算 |

**学習の急所**: 共振周波数 f₀=1/(2π√LC)、Q値=ωL/R 必須暗記。ひずみ波の実効値は各高調波の二乗和の平方根。

## 過渡現象（毎年1〜2問）

**関連テーマ**: [過渡現象](../themes/kato-gensho.md)

| テーマ | 出題頻度 | 形式傾向 |
|---|---|---|
| RC回路の時定数・過渡応答 | ほぼ毎年 | 計算 |
| RL回路の時定数・過渡応答 | 2〜3年に1回 | 計算・論説 |
| テブナン等価回路と過渡現象 | 不定期 | 計算 |
| RLC回路の過渡現象 | 不定期 | 論説 |

**学習の急所**: 時定数τ（RC=CR、RL=L/R）と「t=τで約63%変化」の関係を押さえる。スイッチ切換前後の初期値・最終値を明確にする解法が基本。

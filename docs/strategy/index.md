## 理論科目の基本構造

- **A問題（問1〜14）**: 各5点 × 14問 = 70点。公式・定義・回路計算の理解
- **B問題（問15〜18）**: 各15点（小問a・bに配分） × 4問 = 30点
- **合格ライン目安**: 55点（満点100点）

!!! note "得点戦略の基本方針"
    A問題14問の正答率を上げることが最優先。
    B問題は部分点（小問aだけでも取る）を確実に積み上げる戦術が安定する。
    **1点も落とせないB問題は存在しない** — 難問は捨てて他を固める判断が重要。

## 理論科目の出題傾向（R04〜R07）

| 分野 | 出題スロット | 安定度 |
|------|------------|--------|
| 直流回路（キルヒホッフ・テブナン） | 問5〜7（3問固定） | ◎ 毎回 |
| コンデンサ・静電気 | 問1〜2 | ◎ 毎回 |
| RLC共振 | 問8〜9 | ◎ 毎回 |
| 過渡現象 | 問10 | ◎ 全期固定 |
| 磁気回路・電磁力・インダクタンス | 問3〜4（ローテーション） | △ 2〜3回に1回 |
| 電子理論（半導体・オペアンプ） | 問13〜18（B問題含む） | △ 周期あり |

!!! tip "R08（2026年8月）最優先テーマ"
    [出題周期・波分析](../kakomon/wave-analysis.md) より：
    **磁気回路**（R06上〜R07上 3回連続空白）、**オペアンプ**（R07上空白）、**ブリッジ回路**（R07上空白）
    の3テーマが出題確率最高。[直前3日チェックリスト](last-3days.md) の前日確認に組み込み済み。

```
[計画期 3〜6ヶ月前] 再出題傾向と最適学習戦略 で学習範囲（何年分・何周）を確定

```

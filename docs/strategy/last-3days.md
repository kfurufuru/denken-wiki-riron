## Day 3（試験3日前）— 電磁気・直流回路の公式確認

### 静電気・コンデンサ

!!! note "クーロンの法則 $F = kQ_1Q_2/r^2$ の係数kの根拠"
    $k = 1/(4\pi\varepsilon_0)$。$\varepsilon_0 \approx 8.85 \times 10^{-12}$ [F/m] は真空の誘電率。
    「kを丸暗記する」のではなく、$\varepsilon_0$ から導出できると点が取れる。
    比誘電率 $\varepsilon_r$ がある媒質では $k = 1/(4\pi\varepsilon_0\varepsilon_r)$ と分母が増える（斥力・引力が弱まる）。

**確認事項**:

- [ ] クーロンの法則の距離依存（$r^2$ に反比例）を説明できるか
- [ ] 電界 $E = F/Q$ と電位 $V = kQ/r$ の違いを即答できるか
- [ ] コンデンサの直列合成（逆数の和）と並列合成（足し算）を抵抗と混同しないか → [パターン2★★☆](hikkkake-patterns.md#パターン2)

!!! note "コンデンサのエネルギー $W = \frac{1}{2}CV^2$ の2分の1の根拠"
    充電中の平均電圧は $V/2$（0からVまで線形に上昇）。
    $W = Q \times V_{avg} = CV \times V/2 = CV^2/2$。
    「なぜ1/2か」= 充電開始から完了まで電圧が0→Vと変化するから、平均をとると半分になる。

**確認事項**:

- [ ] $W = Q^2/(2C) = CV^2/2 = QV/2$ の3形式を切り替えられるか → [パターン3★☆☆](hikkkake-patterns.md#パターン3)
- [ ] 直列接続で各コンデンサの電荷が等しい理由を説明できるか

### 磁気・電磁力

!!! note "フレミング左手（モーター）と右手（発電機）の選び方"
    「何が欲しいか」で手を選ぶ。**力Fが欲しい→左手（モーター）**、**起電力eが欲しい→右手（発電機）**。
    左手：中指=電流 I、人差し指=磁束密度 B、親指=力 F（IBF の順）。
    右手：中指=誘起起電力 e、人差し指=磁束密度 B、親指=速度 v（eBv の順）。

**確認事項**:

- [ ] $F = BIL\sin\theta$ の $\theta$ が「電流と磁界の角」であることを確認できるか → [パターン4★☆☆](hikkkake-patterns.md#パターン4)
- [ ] 誘導起電力 $e = BLv\sin\theta$ の $\theta$ が「速度と磁界の角」であることを確認できるか
- [ ] 磁気回路の起磁力 $NI = Hl = \phi R_m$ と電気回路のオームの法則の対応を説明できるか

!!! note "磁気抵抗 $R_m = l/(\mu A)$ の意味"
    電気抵抗 $R = \rho l/A$ と対応。$\mu = \mu_0\mu_r$（鉄心では $\mu_r$ は数百〜数千）。
    空隙では $\mu_r = 1$ なので磁気抵抗が急増する。鉄心区間と空隙区間で材料を必ず確認する。

**確認事項**:

- [ ] 空隙の磁気抵抗 > 鉄心の磁気抵抗（$\mu_r$倍の差）を即答できるか → [パターン5★★☆](hikkkake-patterns.md#パターン5)
- [ ] 環状鉄心の起磁力計算（空隙あり・なし）を手順通り解けるか

### 直流回路

!!! note "テブナンの定理の手順"
    1. 求めたい端子を開放し、開放電圧 $V_{oc}$ を計算
    2. 電源を短絡（電圧源→短絡、電流源→開放）して等価抵抗 $R_{th}$ を計算
    3. $I = V_{oc}/(R_{th} + R_L)$ で負荷電流を求める

**確認事項**:

- [ ] キルヒホッフの電圧則（KVL）・電流則（KCL）の符号を正しく扱えるか
- [ ] 最大電力伝達条件 $R_L = R_{th}$ のときの最大電力 $P_{max} = V_{oc}^2/(4R_{th})$ を導けるか → [パターン9★★★](hikkkake-patterns.md#パターン9)
- [ ] 並列抵抗の合成で「逆数を戻す」ステップを忘れないか → [パターン7★☆☆](hikkkake-patterns.md#パターン7)

## Day 2（試験2日前）— B問題の計算パターン確認

### 三相交流

!!! note "Y結線とΔ結線の√3の出る場所"
    - **Y結線**: 線電圧 = $\sqrt{3}$ × 相電圧 / 線電流 = 相電流（√3なし）
    - **Δ結線**: 線電圧 = 相電圧（√3なし） / 線電流 = $\sqrt{3}$ × 相電流

    **「Y→電圧に√3、Δ→電流に√3」** と整理する。
    混乱したら相回路を描いて、1相分の電圧・電流を基準に考え直す。

**確認事項**:

- [ ] Y-Δ変換時の相電圧・線電圧の関係を図なしで説明できるか → [パターン13★★★](hikkkake-patterns.md#パターン13)
- [ ] 三相電力 $P = \sqrt{3}V_LI_L\cos\phi = 3V_PI_P\cos\phi$ の2形式を使い分けられるか → [パターン14★★☆](hikkkake-patterns.md#パターン14)
- [ ] 三相不平衡時に中性点電流が発生する理由を説明できるか

### RLC直列回路・共振

!!! note "インピーダンス合成はベクトルで行う"
    $Z = \sqrt{R^2 + (X_L - X_C)^2}$。直角三角形のベクトル合成。
    スカラー加算 $Z = R + X_L + X_C$ は絶対に誤り。
    共振条件 $X_L = X_C \Rightarrow \omega L = 1/(\omega C) \Rightarrow \omega_0 = 1/\sqrt{LC}$。

**確認事項**:

- [ ] 直列共振時の電流最大・インピーダンス最小（$Z = R$）を説明できるか
- [ ] Q値（尖鋭度）$Q = \omega_0 L / R$ の大小が帯域幅に与える影響を説明できるか
- [ ] 位相角 $\theta = \arctan\{(X_L - X_C)/R\}$ の符号（誘導性か容量性か）を判定できるか

### 過渡現象

!!! note "スイッチ投入直後のコンデンサ・インダクタの扱い"
    - **コンデンサ（初期電荷ゼロ）**: $t=0^+$ では**短絡**（電圧維持が不可→電流最大）
    - **インダクタ（初期電流ゼロ）**: $t=0^+$ では**開放**（電流維持が不可→電流ゼロ）
    - 定常状態では逆転：コンデンサ→開放（充電完了）、インダクタ→短絡（電流一定）

**確認事項**:

- [ ] 時定数 $\tau = RC$（CR回路）/ $\tau = L/R$（LR回路）を即答できるか
- [ ] $\tau$ 後に「約63%」定常値に達する（≒100%ではない）ことを説明できるか → [パターン19★☆☆](hikkkake-patterns.md#パターン19)
- [ ] 過渡応答の式 $i(t) = I_\infty + (I_0 - I_\infty)e^{-t/\tau}$ を任意の初期値・定常値で使えるか

## Day 1（試験前日）— R08予測テーマの重点確認

### 磁気回路（最優先：R06上〜R07上 3回連続空白）

!!! warning "R08（2026年8月）最高確率テーマ"
    [波分析ページ](../kakomon/wave-analysis.md) より、磁気回路は直近3回連続で出題なし。
    環状鉄心の磁束密度・起磁力計算、またはヒステリシス特性の論述が出題候補。

**確認事項**:

- [ ] 環状鉄心（空隙あり）の計算手順：$NI = H_{\text{iron}}l_{\text{iron}} + H_{\text{gap}}l_{\text{gap}}$ を適用できるか
- [ ] ヒステリシス損 $P_h \propto f B_m^{1.6}$、渦電流損 $P_e \propto f^2 B_m^2$ の周波数依存性を説明できるか
- [ ] 残留磁気・保磁力の意味と用途（ハードとソフト磁性材料の使い分け）を説明できるか

### オペアンプ（高確率：R07上空白）

!!! note "仮想短絡（仮想接地）の適用条件"
    負帰還がかかっているとき、差動入力電圧は限りなくゼロ。
    - 反転増幅: $V_{out} = -\frac{R_f}{R_i}V_{in}$（マイナス符号を忘れない）
    - 非反転増幅: $V_{out} = \left(1 + \frac{R_f}{R_i}\right)V_{in}$
    - 積分回路: 入力を時間積分した出力。$V_{out} = -\frac{1}{RC}\int V_{in}\,dt$

**確認事項**:

- [ ] 反転・非反転増幅のゲイン式を抵抗値から直接導けるか（仮想接地を使った導出で）→ [パターン17★★☆](hikkkake-patterns.md#パターン17)
- [ ] 開ループゲインが有限の場合に「仮想接地が成立しない」条件を説明できるか

### ブリッジ回路（中高確率：R07上空白）

!!! note "ホイートストンブリッジの平衡条件の導出"
    菱形回路で検流計電流ゼロ → 向かい合う辺の積が等しい：$R_1 R_4 = R_2 R_3$。
    「覚える」より「キルヒホッフで計算すると平衡条件が出てくる」流れを追えること。
    平衡時は電源を含む経路で電流が流れ、検流計のみ電流ゼロ（全体がゼロではない）。

**確認事項**:

- [ ] 平衡条件の「向かい合う辺の積」が指す辺を、菱形を描いて確認できるか
- [ ] 「平衡=全電流ゼロ」の誤解を修正できるか（電流は流れる、検流計のみゼロ）
- [ ] 未知抵抗を求める計算（$R_x = R_2 R_3/R_1$）を平衡条件から導けるか

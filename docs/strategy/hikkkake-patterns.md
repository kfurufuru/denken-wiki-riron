# 引っかけパターン集

理論科目で頻出する引っかけパターンを因果説明付きでまとめる。
「なぜ間違いか」を理解することで、同じ誤答を構造的に潰す。

H18〜R07上の過去問から抽出した、繰り返し出る誤答パターンを分野別に整理する。

## 電磁気系

!!! warning "パターン1：電界の向きを「電子が動く向き」と混同"
    ❌ **罠の文章**: 「電子はプラスに引かれるから、電界は−極から+極へ向かう」
    ✅ **正解の考え方**: 電界の向きは「+電荷が受ける力の向き」= **+極（高電位）から−極（低電位）へ**。
    電子は電界の逆向きに動くが、電界そのものの定義は正電荷基準。
    （H23問2・H28問1 類出）

!!! warning "パターン2：コンデンサの直列・並列計算を抵抗と同じと思い込む"
    ❌ **罠の文章**: 「コンデンサ2個直列なら合成容量は $C/2$ になる（抵抗と同じ）」
    ✅ **正解の考え方**: コンデンサは抵抗と**逆転**する。直列 → 逆数の和（小さくなる $1/C = 1/C_1 + 1/C_2$）、並列 → 足し算（$C = C_1 + C_2$）。
    「直列は極板間が広がる（容量低下）→容量小→逆数」と物理で覚える。
    （R04上問1・R06下問2 類出）

!!! warning "パターン3：コンデンサのエネルギーで「2分の1」を忘れる"
    ❌ **罠の文章**: 「電荷 Q、電圧 V のコンデンサのエネルギーは $W = QV$」
    ✅ **正解の考え方**: 充電中の平均電圧は $V/2$（0からVへ線形上昇）。$W = Q \times V/2 = CV^2/2$。
    「エネルギー = 電荷 × 電圧」と単純に覚えると、係数1/2を付け忘れる。
    （H20問3・R05上問1 類出）

!!! warning "パターン4：フレミング左手と右手を逆に使う"
    ❌ **罠の文章**: 「発電機はフレミング左手の法則を使う」
    ✅ **正解の考え方**: 「何が欲しいか」で手を選ぶ。**力Fが欲しい→左手（モーター）**、**起電力eが欲しい→右手（発電機）**。
    試験中に迷ったら「電動機か発電機か」を先に確認する。
    （H25問4・R05下問3 類出）

!!! warning "パターン5：磁気抵抗で $\mu_r$ を忘れて $\mu_0$ だけで計算"
    ❌ **罠の文章**: 「鉄心の磁気抵抗 $R_m = l/(\mu_0 A)$」
    ✅ **正解の考え方**: 鉄心の透磁率は $\mu = \mu_0\mu_r$。$\mu_r$ は数百〜数千。
    空隙のみ $\mu_r = 1$（空気）。鉄心区間と空隙区間で材料を区別することが必須。
    （H29問4・R06上問3 類出）

!!! warning "パターン6：相互インダクタンスMの合成インダクタンスで符号を間違える"
    ❌ **罠の文章**: 「2コイルの合成インダクタンスは常に $L_1 + L_2 + 2M$」
    ✅ **正解の考え方**: 巻き方（同名端子の向き）で符号が決まる。和動接続 → $L_1 + L_2 + 2M$、差動接続 → $L_1 + L_2 - 2M$。同名端子マーク（•）の向きを確認する。
    （H24問3・R07上問4 類出）

## 直流回路系

!!! warning "パターン7：並列合成抵抗で「逆数に戻すのを忘れる」"
    ❌ **罠の文章**: 「$1/R_p = 1/4 + 1/4 = 1/2$ だから $R_p = 1/2\,\Omega$」
    ✅ **正解の考え方**: $1/R_p = 1/2$ を逆数に戻して $R_p = 2\,\Omega$。
    計算の最後ステップで必ずひっくり返す。2素子の場合の「積÷和」公式を使うと検算しやすい。
    （H18問5・R04下問5 類出）

!!! warning "パターン8：テブナンの等価回路で電源短絡・開放の操作を間違える"
    ❌ **罠の文章**: 「等価内部抵抗を求めるとき、電流源を短絡する」
    ✅ **正解の考え方**: 電圧源 → 短絡（内部抵抗ゼロの電池として扱う）、電流源 → 開放（内部インピーダンス無限大として扱う）。逆の操作をすると等価回路が完全に変わる。
    （H27問6・R05上問6 類出）

!!! warning "パターン9：最大電力伝達で「効率最大」と「電力最大」を混同"
    ❌ **罠の文章**: 「$R_L = R_{th}$ のとき電力効率が最大になる」
    ✅ **正解の考え方**: $R_L = R_{th}$ のときは**負荷電力が最大**になる条件。このとき電源の半分の電力が内部抵抗で消費されるため、効率は50%にすぎない。「電力最大≠効率最大」。
    （H22問7・R06下問5 類出）

## 交流回路系

!!! warning "パターン10：インピーダンス合成をスカラー加算してしまう"
    ❌ **罠の文章**: 「直列RLC回路で $Z = R + X_L + X_C$」
    ✅ **正解の考え方**: RとリアクタンスはベクトルでZを合成。$Z = \sqrt{R^2 + (X_L - X_C)^2}$。
    「直角三角形の斜辺」がZであることを図で確認する。スカラー加算は実部同士・虚部同士にしか使えない。
    （H19問9・R04上問8 類出）

!!! warning "パターン11：コイルとコンデンサの位相関係を逆に覚える"
    ❌ **罠の文章**: 「コイルは電流が電圧より進む」
    ✅ **正解の考え方**: **ELI the ICEman** が定番の因果整理。L（コイル）は電圧E→電流I（電流遅れ）、C（コンデンサ）は電流I→電圧E（電流進み）。「コイルに電圧をかけると電流が追いかけてくる遅れ」と物理で理解する。
    （H21問8・R05下問9 類出）

!!! warning "パターン12：実効値と最大値を混同する"
    ❌ **罠の文章**: 「コンセント100Vが最大値だ」
    ✅ **正解の考え方**: 100Vは**実効値**。最大値は $100\sqrt{2} \approx 141\,\text{V}$。
    絶縁設計では最大値を使う。実効値は「等しい仕事量をする直流値」として定義される（正弦波では $V_{max}/\sqrt{2}$）。
    （H18問8・R06上問8 類出）

## 三相交流系

!!! warning "パターン13：Y結線とΔ結線の√3の出る場所を混同する"
    ❌ **罠の文章**: 「Y結線でも線電流 = $\sqrt{3}$ × 相電流になる」
    ✅ **正解の考え方**: Y結線は **線電流 = 相電流**（√3なし）、**線電圧 = $\sqrt{3}$ × 相電圧**。Δ結線は **線電圧 = 相電圧**（√3なし）、**線電流 = $\sqrt{3}$ × 相電流**。「Y→電圧に√3、Δ→電流に√3」と整理する。
    （H20問12・R05上問13 類出）

!!! warning "パターン14：三相電力の計算で√3を忘れる"
    ❌ **罠の文章**: 「三相電力は単相電力 $P_1 = V_PI_P\cos\phi$ の3倍だから $P = 3V_PI_P\cos\phi$、線間電圧で書き直すと $P = 3V_LI_L\cos\phi$」
    ✅ **正解の考え方**: Y結線では $V_P = V_L/\sqrt{3}$ なので代入すると $P = 3 \times (V_L/\sqrt{3}) \times I_L\cos\phi = \sqrt{3}V_LI_L\cos\phi$。$3/\sqrt{3} = \sqrt{3}$ になる。$3V_LI_L\cos\phi$ は誤り。
    （H23問13・R07上問11 類出）

## 電子理論系

!!! warning "パターン15：N型半導体が「負に帯電している」と誤解する"
    ❌ **罠の文章**: 「N型はNegativeの略だから、物質全体が負電荷を持っている」
    ✅ **正解の考え方**: N型の「N」は「多数キャリアが電子（Negative）」を意味するだけ。不純物のリン原子は+5価の核と4個の共有結合で安定し、余剰電子1個を放出後は正イオン化する。物質全体の電荷は**中性**。
    （H24問14・R06上問13 類出）

!!! warning "パターン16：逆バイアスで電流が完全にゼロになると思い込む"
    ❌ **罠の文章**: 「ダイオードに逆バイアスをかけると電流はゼロになる」
    ✅ **正解の考え方**: 逆バイアスでも微小な**逆飽和電流 $I_s$** が流れる。温度依存性が高く、温度上昇で急増する。ツェナー降伏電圧に達すると急激に大電流が流れる（ツェナーダイオードはこれを利用）。
    （H26問14・R05下問13 類出）

!!! warning "パターン17：オペアンプの反転増幅ゲインにマイナスをつけ忘れる"
    ❌ **罠の文章**: 「反転増幅回路のゲインは $A_v = R_f/R_i$」
    ✅ **正解の考え方**: 反転増幅は出力が入力と逆位相になるため $A_v = -R_f/R_i$。問題文が「ゲインの絶対値」を問うなら $|A_v| = R_f/R_i$ でよいが、出力電圧を求める場合はマイナス符号を含めないと符号が逆になる。
    （H28問18・R06下問17 類出）

## 過渡現象系

!!! warning "パターン18：コンデンサをt=0直後に「開放」として扱う"
    ❌ **罠の文章**: 「コンデンサは電荷を蓄えるから、スイッチ直後は電流が流れない→開放として扱う」
    ✅ **正解の考え方**: **初期電荷ゼロ**のコンデンサは $t=0^+$ で**短絡**（電圧=0を維持しようとするため電流が最大値で流れ始める）。逆に定常状態では充電完了し電流ゼロ→開放。スイッチ直後と定常後で扱いが逆転する。
    （H25問10・R04上問10 類出）

!!! warning "パターン19：時定数後に「完全に定常値に達した」と思い込む"
    ❌ **罠の文章**: 「時定数τ後に回路は定常状態になる」
    ✅ **正解の考え方**: τ後には定常値の約63%（より正確には $1-1/e \approx 63.2\%$）に達するだけ。5τ後にようやく99%（≈定常値）。「τ=100%」は誤り。試験では「τ秒後の電流・電圧は？」が問われることがあり、式 $i(\tau) = I_\infty(1 - e^{-1})$ で計算する。
    （H22問10・R05上問10 類出）

## 計測系

!!! warning "パターン20：可動コイル型計器が交流でも使えると思い込む"
    ❌ **罠の文章**: 「可動コイル型は電流の大きさに反応するから、交流でも使える」
    ✅ **正解の考え方**: 可動コイル型は**直流専用**。永久磁石の磁界と電流の向きで力が発生する仕組み。交流では電流の向きが毎サイクル反転し、力も反転→指針が振れずゼロを示す。交流測定には可動鉄片型・整流型を使う。
    （H19問15・R06下問14 類出）

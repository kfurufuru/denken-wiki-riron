# 過去問逆引きパス — 間違えた問題からテーマを特定する

過去問を解いて間違えた問題を、対応テーマに紐付けて集中学習するためのパス。
「どこを直せばいいか」を素早く特定し、無駄なく弱点を潰す。

---

## 弱点パターン別 逆引きルート

「○○が苦手」という症状から、原因・対策・参照ページを直接引ける逆引きマップ。
症状に当てはまるものを選んで対策ページへ進む。

---

## 「交流の位相・フェーザーが苦手」な人向けルート

### 症状: 位相差の±符号でいつも迷う

**原因**: ELIのICE（コイルは電圧が進む、コンデンサは電流が進む）を丸暗記しているが、
フェーザー図との対応で「進む＝角度が大きい＝正」なのか混乱している。

**対策**:
1. [交流回路基礎](../themes/kouryu-kiso.md) を開く
2. 「フェーザー図の読み方」節で「反時計回りが正の位相（進み）」を確認する
3. ELIのICEを図で確認する（暗記ではなく「なぜそうなるか」を原理から追う）
4. 確認問題：コイル単体の回路でi(t)とv(t)の位相差を求める問題を1問解く

!!! tip "判定基準"
    フェーザー図で「基準ベクトルより反時計回り＝進み位相＝正の位相角」。
    「進む」「遅れる」は常に電圧を基準に表現される点に注意。

---

## 「RLC回路のインピーダンス計算が苦手」な人向けルート

### 症状: インピーダンス計算でスカラー加算してしまう（Z = R + XL + XCと計算する）

**原因**: 抵抗・コイル・コンデンサのインピーダンスをベクトル量ではなくスカラー量として扱っている。
RとXL、XCは90°位相がずれているため、単純加算は不可。

**対策**:
1. [RLC回路](../themes/rlc-kairo.md) を開く
2. 「インピーダンスの合成」節でベクトル図（直角三角形）を確認する
3. Z = √{R² + (XL − XC)²} の導出を追う（ピタゴラスの定理の応用であることを確認）
4. 直列RLC回路で数値を入れて Z を計算する練習問題を2問解く

!!! warning "絶対に誤りの式"
    Z = R + XL + XC はスカラー加算であり、**常に誤り**。
    R と (XL − XC) は互いに直交するベクトルなので、合成はピタゴラスで行う。

---

## 「過渡現象の初期値・最終値の設定が苦手」な人向けルート

### 症状: スイッチ投入直後（t=0⁺）と定常状態（t=∞）の設定が混乱する

**原因**: コンデンサとインダクタの「初期状態での振る舞い」と「定常状態での振る舞い」が
反転することを理解していない。

**対策**:
1. [過渡現象](../themes/kato-gensho.md) を開く
2. 「初期条件と最終値の設定」節で下の2規則を確認する
3. 練習問題でt=0⁺の等価回路とt=∞の等価回路を別々に描く訓練をする

| 素子 | t=0⁺（初期電荷/電流ゼロの場合） | t=∞（定常状態） |
|------|-------------------------------|-----------------|
| コンデンサ | 短絡（電流が流れる） | 開放（充電完了） |
| インダクタ | 開放（電流が流れない） | 短絡（電流一定） |

!!! tip "覚え方ではなく理解"
    コンデンサは「電圧を急変できない」→ t=0⁺では電圧ゼロ＝短絡と等価。
    インダクタは「電流を急変できない」→ t=0⁺では電流ゼロ＝開放と等価。

## 「三相交流の√3がわからない」な人向けルート

### 症状: Y結線かΔ結線かで√3をどこにかけるか毎回混乱する

**原因**: 公式を「Y結線→VL=√3Vp」のように暗記しているが、なぜ√3が出るかの
導出を追っていないため、設問で電圧か電流かを問われると混乱する。

**対策**:
1. [三相交流](../themes/sansou-kouryu.md) を開く
2. 「Y結線のフェーザー図」節で線間電圧が√3倍になる幾何学的な理由を確認する
3. 「Y→電圧に√3、Δ→電流に√3」の対称性を確認する
4. 三相電力の2形式（P = √3 VLIL cosφ と P = 3VPIPcosφ）が等価であることを導く

| 結線 | 電圧の関係 | 電流の関係 |
|------|----------|----------|
| Y結線 | 線電圧 = **√3** × 相電圧 | 線電流 = 相電流 |
| Δ結線 | 線電圧 = 相電圧 | 線電流 = **√3** × 相電流 |

!!! tip "確認の仕方"
    「√3はどこか？」→ Y結線なら「電圧側」、Δ結線なら「電流側」と即答できるまで反復。

## 「コンデンサと直流の関係が混乱する」な人向けルート

### 症状: コンデンサに直流を加えたとき「開放」なのか「短絡」なのか、どちらか迷う

**原因**: 定常状態（t=∞）と過渡状態（t=0⁺）の区別ができていない。
「直流では」という条件が、どの時刻の話かによって答えが正反対になる。

**対策**:
1. [コンデンサ](../themes/condenser.md) を開く
2. 「直流回路でのコンデンサの扱い」節を確認する
3. [過渡現象](../themes/kato-gensho.md) の「初期条件と最終値の設定」節と合わせて読む
4. 確認問題：RC直列回路のスイッチ投入直後と十分時間後の電流を求める

| 状況 | コンデンサの扱い |
|------|----------------|
| 定常状態（直流が十分流れた後） | **開放**（充電完了で電流ゼロ） |
| スイッチ投入直後（初期電荷ゼロ） | **短絡**（電圧がゼロ、電流最大） |
| 交流回路 | インピーダンス XC = 1/(2πfC) をもつ素子 |

## 「トランジスタのhパラメータ等価回路が読めない」な人向けルート

### 症状: hパラメータ等価回路が何を表しているのかわからず、利得の計算ができない

**原因**: hパラメータ（hie, hfe, hre, hoe）の物理的な意味と等価回路中での
各素子の役割を対応させないまま計算しようとしている。

**対策**:
1. [トランジスタ](../themes/transistor.md) を開く
2. 「hパラメータ等価回路の読み方」節で各パラメータの定義を確認する
3. hieが入力インピーダンス、hfeが電流増幅率であることを確認する
4. 電流増幅率 Ai = hfe の近似と、電圧利得 Av = -hfe × RL / hie の導出を追う

| パラメータ | 意味 | 等価回路での役割 |
|-----------|------|----------------|
| hie | 入力インピーダンス [Ω] | ベースとエミッタ間の抵抗 |
| hfe | 電流増幅率（無次元） | 電流制御電流源 hfe × IB |
| hre | 電圧帰還率（小さく無視可） | 通常ゼロ近似 |
| hoe | 出力アドミタンス（小さく無視可） | 通常開放近似 |

!!! note "試験での使い方"
    電験3種ではhreとhoeは無視（ゼロ近似）して計算する場合がほとんど。
    「hie = 入力抵抗、hfe = 電流増幅率」の2つだけ確実に押さえる。

## 「磁気回路と電気回路の類比がわからない」な人向けルート

### 症状: 磁気回路の問題で何をどう計算すればいいか、出発点がわからない

**原因**: 磁気回路の物理量（φ, NI, Rm）を電気回路（I, V, R）と対応させていないため、
どの公式を使えばいいかが判断できない。

**対策**:
1. [磁気回路](../themes/jiki-kairo.md) を開く
2. 「電気回路との類比」節で対応表を確認する
3. 環状鉄心の起磁力計算を1題、オームの法則の電気版と並べて解いてみる

| 電気回路 | 磁気回路 | 対応 |
|---------|---------|------|
| 電流 I [A] | 磁束 φ [Wb] | 流れるもの |
| 起電力 V [V] | 起磁力 NI [A] | 駆動力 |
| 抵抗 R [Ω] | 磁気抵抗 Rm [A/Wb] | 流れにくさ |
| オームの法則 V = IR | NI = φ × Rm | 同じ構造 |

!!! tip "解き方の手順"
    磁気回路の問題は「電気回路の問題に置き換えて解く」だけ。
    Rm = l/(μA) で磁気抵抗を求めたら、あとは電気回路と同じ手順で計算できる。

## 「Q値の式を直列・並列で間違える」な人向けルート

### 症状: Q値の公式を直列と並列で混同して計算ミスする

**原因**: 「Q = ωL/R」と「Q = R/ωL」のどちらが直列でどちらが並列かを
暗記しようとしており、試験中に混乱する。

**対策**:
1. [RLC回路](../themes/rlc-kairo.md) を開く
2. 「Q値と選択度」節でQ値の定義（エネルギー蓄積/消費の比）から導出を確認する
3. 直列共振と並列共振の回路図を描いてQ値の意味の違いを確認する

| 回路 | Q値の式 | 物理的意味 |
|------|---------|-----------|
| 直列RLC | Q = ωL/R = 1/(ωCR) | 電流が最大のとき電圧拡大率 |
| 並列RLC | Q = R/(ωL) = ωCR | 電圧が最大のとき電流拡大率 |

!!! tip "混同しない方法"
    「直列のQ」は「Rが分母（Rが大きいと損失が多くQが下がる）」。
    「並列のQ」は「Rが分子（Rが大きいと電流が分散しにくくQが上がる）」。
    公式の形より、直並列での「Rの役割の逆転」を理解すると間違えにくい。

## 「オペアンプの仮想接地をいつ使えばいいかわからない」な人向けルート

### 症状: 仮想短絡（仮想接地）を適用するタイミングがわからず、回路解析が止まる

**原因**: 「負帰還がかかっているとき」という適用条件を理解せず、
すべてのオペアンプ回路に仮想接地を使おうとしている。

**対策**:
1. [オペアンプ](../themes/opamp.md) を開く
2. 「仮想接地の適用条件」節で「負帰還がある」「オープンループでない」の2条件を確認する
3. 反転増幅・非反転増幅・積分回路の3つで「どこが仮想接地か」を図で確認する
4. ゲイン式を仮想接地を使った導出で求め、覚えるのではなく「導ける」状態にする

**仮想接地の適用手順**:

1. 負帰還があるか確認（出力から入力の－端子への帰還経路があるか）
2. ある場合：V+ = V−（差動入力電圧＝0）と置く
3. ない場合：仮想接地は使えない（オープンループ動作）

!!! note "電験3種の試験では"
    出題されるオペアンプ回路はほぼすべて負帰還あり。
    仮想接地を使ってKCLを立てれば、ゲイン式は必ず導出できる。

## 「力率改善の計算でtanφとsinφを使い分けられない」な人向けルート

### 症状: 力率改善に必要なコンデンサ容量を求めるとき、tanφかsinφかで迷う

**原因**: 有効電力・無効電力・皮相電力の3角形（電力三角形）を図で確認せず、
「Q = P tanφ」と「Q = S sinφ」の使い分けが体で染みついていない。

**対策**:
1. [交流電力](../themes/kouryu-denryoku.md) を開く
2. 「電力三角形」節で P（底辺）・Q（垂直）・S（斜辺）の関係を確認する
3. cosφ=0.6→0.9への改善で必要なQ'（補償無効電力）の計算手順を追う

**力率改善の計算ルート**:

```
改善前: tanφ1 = Q1/P, sinφ1 = Q1/S1
改善後: tanφ2 = Q2/P
補償無効電力: ΔQ = Q1 − Q2 = P(tanφ1 − tanφ2)
コンデンサ容量: C = ΔQ/(ωV²) または ΔQ = V²ωC
```

!!! tip "tanφとsinφのどちらを使うか"
    有効電力Pが既知のとき → **tanφを使う**（Q = P tanφ）。
    皮相電力Sが既知のとき → **sinφを使う**（Q = S sinφ）。
    力率改善の問題では有効電力Pが与えられることが多いのでtanφが主役。

## このパスの使い方

1. 過去問または模擬試験を解く
2. 間違えた問題の「問題のキーワード・内容」を確認する
3. 下の対応表から「参照すべきテーマ」を特定する
4. そのテーマの詳細ページで該当箇所を復習する
5. 同じタイプの問題をもう1問解いて定着確認

!!! tip "e-logとの連携"
    間違えた問題は `.secretary/denken-study/e-log.md` に記録しておく。
    「誤答記録」コマンドで L1（知識エラー）/ L2（認知バグ）に分類することで、
    「知識が足りない」のか「理解しているのに計算ミスした」のかを区別できる。

## 試験まで1ヶ月の場合の絞り込み戦略

時間が限られている場合は、出題頻度×自分の正答率で優先順位をつける。

### ステップ1: 現状の正答率を把握する

最近解いた過去問で、テーマ別の正答率を出す（大まかで良い）。

### ステップ2: 優先マトリクスで分類する

|  | 正答率が高い（60%以上） | 正答率が低い（60%未満） |
|---|---|---|
| **出題頻度が高い** | 維持（軽くおさらい） | **最優先で強化** |
| **出題頻度が低い** | 放置でOK | 捨て科目候補 |

### ステップ3: 最優先テーマだけ確認する

「最優先で強化」に入ったテーマのみ、詳細ページと過去問を集中的にこなす。

!!! warning "捨て科目の基準"
    残り期間が1ヶ月以内で、正答率が低く出題頻度も低いテーマは「捨てる」判断もあり得る。
    ただし、捨てるのは「確実に得点できるテーマが6割以上カバーできている」場合に限る。

### 最優先テーマ（試験1ヶ月前に確保すべき5テーマ）

1. [三相交流](../themes/sansou-kouryu.md) — 毎年出る・配点高い
2. [RLC回路](../themes/rlc-kairo.md) — 共振問題は計算パターンが固定
3. [直流回路](../themes/chokuryu-kairo.md) — 基礎固め直し
4. [交流電力](../themes/kouryu-denryoku.md) — 力率改善は公式適用で得点できる
5. [過渡現象](../themes/kato-gensho.md) — 時定数さえ覚えれば計算は速い

## よくある「間違えのパターン」別チェックリスト

### 計算は合っているのに答えが違う

- [ ] 単位換算を間違えていないか（kΩ → Ω, μF → F）
- [ ] 実効値と最大値を取り違えていないか
- [ ] √3 を掛け忘れ・割り忘れていないか（三相）
- [ ] 逆数に戻し忘れていないか（並列合成・直列コンデンサ）

### 公式は知っているのに適用できない

- [ ] 問題の回路図を正しく読めているか（Y/Δの区別）
- [ ] どの値が「相量」でどの値が「線量」か確認したか
- [ ] フェーザー図を描いて位相関係を確認したか

### そもそも解き方が分からない

→ 対応テーマの詳細ページで「典型問題パターン」を確認する

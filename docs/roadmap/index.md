# 学習ロードマップ

電験3種 理論科目を効率よく攻略するための3つの学習コースです。
目的に合わせてコースを選んでください。

---

## コース一覧

| コース | 対象 | 期間目安 |
|---|---|---|
| [初学者パス](beginner.md) | 理論をゼロから始める方 | 3〜4ヶ月 |
| [過去問逆引き](by-weakness.md) | 弱点テーマを集中強化したい方 | 随時 |
| [直前確認パス](last-minute.md) | 試験2〜4週前の総仕上げ | 2〜4週間 |

---

## テーマ依存関係（Phase 1〜4）

理論16テーマには学習上の依存関係があります。
以下のフェーズ順に進めると、前提知識が積み上がった状態で各テーマに入れます。

### Phase 1 — 電気回路の基礎（必須）
依存関係なし。最初に固める土台。

```
直流回路 → 交流回路基礎 → RLC回路 → 交流電力 → 三相交流
```

### Phase 2 — 電磁気（Phase 1 が前提）
静電気・コンデンサは直流回路の理解があると理解しやすい。
電磁力・磁気回路・インダクタンスは交流回路基礎の知識を使う。

```
静電気 → コンデンサ
電磁力 → 磁気回路 → インダクタンス
```

### Phase 3 — 過渡現象（Phase 1 + Phase 2 が前提）
RLC回路・インダクタンス・コンデンサの理解が必要。

```
（RLC回路 + インダクタンス + コンデンサ） → 過渡現象
```

### Phase 4 — 電子理論・電気計測（Phase 1〜3 が推奨）
計器・ブリッジは直流・交流回路の知識で対応可能。
半導体・トランジスタ・オペアンプは独立性が高い。

```
半導体 → トランジスタ・FET → オペアンプ
直流回路 → ブリッジ回路
交流回路基礎 → 計器の原理
```

---

!!! tip "学習効率を上げるコツ"
    各テーマを学んだ後は「フェインマン [テーマ名]」コマンドで理解度を確認。
    スコア4以上になったら次のテーマへ進む。
